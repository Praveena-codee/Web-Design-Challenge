Weather API's Project

The purpose of this assignment was to apply our recently acquired knowledge to use API’s and process its data into significant results that can be interpreted in order to provide relevant insights.

In this project we connected to the OpenWeatherMaps API and collect the data from over 500 cities, gathering data such as Temperature, Humidity, Cloudiness and Wind speed. Then we process the data through with the use of the Pandas Library and plot through MatPlotLib to obtain meaningful and visual insights.

Then we run some lineal regression on the data sets to determine the level of correlation between the latitude and the specific measured parameter. Some brief conclusions are shown in each “Plots page”.
